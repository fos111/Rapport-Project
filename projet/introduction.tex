\markboth{Introduction Générale}{}


 Le projet « Inventaire intelligent» consiste à concevoir et développer une solution IoT intégrée pour la gestion automatisée des stocks.
Chaque produit sera identifié par un code QR unique, scanné par un module IoT (ESP32) connecté à une application Web  et Mobile permettant de suivre et de gérer les produits en temps réel.
Ce système vise à faciliter la gestion des inventaires, réduire les erreurs humaines et améliorer la traçabilité grâce à une interface web ergonomique et une architecture connectée.
